❓ Häufig gestellte Fragen

🤖
Welche Bot-Sprachen werden unterstützt?
▼
Wir unterstützen alle gängigen Programmiersprachen für Discord Bots: Node.js (discord.js), Python (discord.py), Java (JDA), C# (Discord.NET), Go (DiscordGo) und viele mehr. Du kannst auch eigene Umgebungen einrichten.
⚡
Wie schnell startet mein Bot nach einem Crash?
▼
Unser Monitoring-System erkennt Bot-Crashes innerhalb von Sekunden und startet deinen Bot automatisch neu. In der Regel ist dein Bot binnen 10-30 Sekunden wieder online.
📁
Kann ich mehrere Bots auf einem Paket hosten?
▼
Ja, solange die Ressourcen (RAM, CPU) ausreichen, kannst du mehrere Bots parallel laufen lassen. Wir empfehlen das Standard oder Premium Paket für mehrere Bots.
🔄
Wie lade ich meinen Bot-Code hoch?
▼
Du kannst deinen Code per SFTP, Git Clone oder über unser Web-Interface hochladen. Wir unterstützen auch direkte GitHub/GitLab Integration für automatische Deployments.
💾
Werden meine Bot-Daten gesichert?
▼
Ja, alle Bot-Daten werden täglich automatisch gesichert. Du kannst auch jederzeit manuelle Backups erstellen und alte Versionen wiederherstellen.
🎵
Kann ich Musik-Bots hosten?
▼
Absolut! Unsere Server sind perfekt für Musik-Bots geeignet. Mit ausreichend Bandbreite und CPU-Power für YouTube, Spotify und andere Audio-Quellen.
